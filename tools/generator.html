2 Select Modes

expand_more
Choose which controllers to include. Each selected mode will appear in the remote's menu carousel and get its own entity configuration section below.
ac_unit AC
lightbulb Lights
roller_shades Covers
bolt Automations
settings Settings

ac_unit AC Units 0

expand_more
Add one entry per HA climate.* entity. Enter the Entity ID exactly as it appears in Home Assistant (e.g. climate.living_room_ac) and a short Display Name shown on the remote screen (max 8 chars, enforced). Then pick the HVAC Modes and Fan Modes your unit supports - the remote will cycle through them in the order you select them.

lightbulb Lights 0

expand_more
Add one entry per HA light.* entity. Enable Dimmable for lights that support brightness (0-100 %) control.

roller_shades Covers 0

expand_more
Add one entry per HA cover.* entity (curtain, blind, roller, etc.).

bolt Automations 0

expand_more
Add scenes, automations, scripts and toggles. Up to two can be pinned as bottom-bar shortcuts via Menu Slot L / R. Short Name (max 5 chars) is shown on the shortcut button.
